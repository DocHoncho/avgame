# AVGame

A Tron-inspired isometric action game with twin-stick shooter mechanics and ARPG progression.

## Tech Stack

- **TypeScript** – primary language
- **Vite** – build system & dev server with hot‑reload
- **Three.js** – 3‑D graphics engine
- **Vue 3** – UI/HUD framework (native Vite integration)
- **XState** - State management

## Getting Started

### Prerequisites

- Node.js 16+
- npm or pnpm

### Installation

```bash
# Install dependencies
npm install
# or
pnpm install
```

### Development

```bash
# Start dev server with hot-reload
npm run dev
# or
pnpm dev
```

### Build

```bash
# Build for production
npm run build
# or
pnpm build
```

## Project Structure

```
./              # project root
├─ src/         # source code
│  ├─ engine/   # game engine components
│  ├─ state/    # state management
│  └─ ui/       # Vue components
├─ public/      # static assets
└─ docs/        # technical & design docs
```
